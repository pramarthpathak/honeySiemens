Important Information about Installing and Working with the IT-CP

The filesystem has the following structure:

Folder Significance
/
(standard home page)		 The folder contains a predefined home page for the CP that can be adapted to suit your own requirements. For this home page, you require the files in the folders /SYS and /home.
/home The folder contains the files of the predefined home page of the CP. If necessary the files may be deleted.
/SYS
(read-only)		 The folder contains important read-only system pages and HTML pages specific to the plant that were created during configuration.
/applets
(do not delete)		 The folder contains the files "s7applets.jar, s7api.jar, s7gui.jar, s7util.jar, s7misc.jar" that are necessary when using S7 applets.
/user
(your HTML pages)		 This folder contains user-defined HTML pages. If this folder contains a page called "index.htm", this page will be selected and displayed when only the IP address of the CP is specified in a browser, for example http://142.11.39.172 -> http://142.11.39.172/user/index.htm
/config
(do not delete)		 The folder contains the FTP client configuration file "file_db.txt". The file "S7wmeld.edb" contains the text for page "Online S7 Diagnostics".
/examples The folder contains HTML files with java applet and java script examples - may be deleted.
/ram
(do not delete)		 The folder contains temporary files.
